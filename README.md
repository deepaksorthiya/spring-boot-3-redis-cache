# Getting Started
## Requirements:
```
Spring Boot : 3.2.2
Java : 17
Redis Server : 7
Docker Engine: 25.0.2
Compose: v2.24.3-desktop.1
```
Set up a Docker local environment on your machine by running these commands in the terminal:

1. Start Redis Server:
    ```bash
    docker compose up
   ```
2. Start Spring boot app:
    ```bash
    mvn spring-boot:run
